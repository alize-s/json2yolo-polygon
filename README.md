# json2yolo-polygon
A tool that converts JSON in LabelMe format to TXT in Yolo.
一个可以将labelme标注多边形后生成的json文件转换为txt的小工具。
![image](https://github.com/user-attachments/assets/8e5a8ec2-4a4a-4d6d-96f8-49c6ab487061)
使用GUI界面时有必须要输入的几个参数：
①存放json文件的目录。
②要存放转换后文件的目录。
③标注文件内的标签信息。标签之间以逗号分隔。标签按照labelme标注的顺序填写。
④图像的宽度和高度。
填写完成后点击开始即可。
